# Banking Analytics Dashboard

End-to-end Banking Analytics Project built from scratch using SQL Server, NoSQL, and Power BI. Designed relational and NoSQL datasets, wrote SQL queries manually, and created interactive dashboards for loan performance, branch insights, and complaint resolution analytics. AI-generated data was used for practice and analysis.

## Tools
- SSMS / SQL Server
- NoSQL database
- Power BI

## Features
- Loan application analysis
- Funded and outstanding amount tracking
- Branch-wise loan insights
- Complaint and resolution analysis
- Raw event / complaint data using NoSQL
